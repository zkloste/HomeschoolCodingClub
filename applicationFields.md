# Homeschool Coding Club Application
## Account Requirement
- A parent/guardian must create the account.
- A parent/guardian must complete one application for each student who wants to join.
---
## Section 1: Parent/Guardian Information
- Parent/guardian full name
- Relationship to student
- Email address
- Phone number
- Preferred contact method (email/text/phone)
---
## Section 2: Household Application Details
- Number of children applying
- Primary home city/area
- Preferred schedule options (weekday daytime, weekday evening, weekend)
---
## Section 3: Student Information (repeat for each child)
- Student full name
- Age
- Grade level

### Student technology access
- device available (laptop/desktop/tablet)
    - need to share with a sibling?
- operating system (windows, mac, chromebook)

### Student Background
- Has the student done coding before? (yes/no)
- If yes, what have they used? (Scratch, Python, JavaScript, robotics kits, etc.)
- Comfort level (beginner / some experience / intermediate)
### Student Motivation
- In the student’s own words: Why do you want to join this coding club?
  - Club motto: “Build cool stuff.”
- What do you hope to build or learn in the next semester?



### Interest Areas
- for each child, rate the following categories in order of interest (5 stars = most interested):
	1 - build with hardware (robots, sensors, and physical computing)
		- custom small scale smart home (sensors to monitor/control air temp, humidity, door security, etc.)
		- automated hydroponics garden (grow lettuce indoors with LED and nutrient monitoring)
		- hand controlled rc car (program a glove that detects hand movements to control an rc car)
	2 - make games (design and code playable games)
		- create a 2d arcade game (pong, snake, tetris, or original game)
		- puzzle/escape room game (logic challenges and branching clues)
		- interactive adventure game (choose-your-own-story with branching paths and endings)
	3 - ai/ml projects (build smart tools with artificial intelligence)
		- custom ai chatbot website (tutorAI, storyCreatorAI, talkLikeAPirateAI)
		- image classifier (train a model to recognize objects or hand signs)
		- ai quiz generator (upload notes/textbook and generate study quizzes)
	4 - creative coding (art, music, storytelling, and interactive media)
		- meme and sticker studio app (create captions, remix templates, and share custom designs)
		- avatar and character creator website (mix hairstyles, outfits, colors, and accessories, then save/share designs)
		- music mood visualizer (live visuals that react to sound)
	5 - real-world impact projects (solve school or community problems)
		- recycling/litter tracking app (log cleanups and progress)
		- smart classroom tool (team generator, attendance helper, live polls)
		- community donation finder app (map local food pantries, shelters, and clothing drop-off sites)
